## 二、 顶级交互效果的技术实现方案 (Tech Stack & Implementation)

为了让咱们的 Next.js 架构能完美支撑这种苹果级交互，我们需要在不破坏原有部署优势（Cloudflare Pages 零成本托管）的前提下，对前端武器库进行全面升级：

**代码段**

```
graph TD
  A[用户的鼠标/手指滚动] -->|驱动| B(Lenis.js 平滑物理滚动库)
  B -->|时间线映射| C{GSAP ScrollTrigger 核心引擎}
  C -->|进度 0% - 100%| D[HTML5 Canvas 序列帧渲染引擎]
  C -->|Z-index 错位| E[大号艺术字体与空间浮动元素]
  F[右下角常驻挂件] -->|点击触发| G[原生/Canvas 粒子喷射系统 💖 🐟]
```

### 1. 核心底层：Next.js + Cloudflare Pages（保留原有优势）

* 继续沿用 `next build && next export` 静态打包技术。
* **零成本服务器：** 哪怕网页放了几百张序列帧图片，利用 Cloudflare 全球极速 CDN 分发，我们依然不需要付一分钱的带宽和流量费，保障副业  **近乎 100% 的净利润率** 。

### 2. 动画时间线总管：GSAP (GreenSock) + `@gsap/react`

* 这是替代原生 React 动画的重武器。我们将引入 `gsap` 和 `ScrollTrigger` 插件。
* **实现原理：** 将整个页面的滚动距离（比如向下滚动 2000px）绑定为一个动作时间线。当滚动到 20% 时，触发小猫“从包里探出头”；滚动到 50% 时，背景“DANG DANG”文字以 0.5 的视差速度慢速升起，同时触发右侧打字机动效。

### 3. 苹果式核心交互：HTML5 Canvas 序列帧渲染 (Sequence Frame Scrubbing)

* **绝对不要直接放 MP4 视频！** 视频无法随着手指的上下往复滚动而精准倒放和前进。
* **工程做法：** 1. 在页面放置一个全屏的 `<canvas>` 标签。
  2. 用 JS 预加载 60~120 张连续的透明 WebP 动作图片入内存。
  3. 利用 `ScrollTrigger` 将当前滚动百分比（0.00 to 1.00）实时映射为数组下标（第 1 到第 120 张图）。
  4. 每次滚动时，通过 `ctx.drawImage()` 高速刷新画布。**这就实现了视频中“随手划一下，小猫就动一下”的惊艳视差效果！**

### 4. 丝滑滚动底座：Lenis.js

* 浏览器原生的滚动条是“跳跃式”的，直接绑定序列帧会显得非常卡顿和僵硬。
* 引入 `Lenis` 平滑滚动库，赋予鼠标滚轮类似于手机屏幕般的“物理惯性与极润阻尼感”，这是让高级网页展现出奢华质感的关键秘密。

## 三、 前置资产与素材标准化清单 (Asset Pipeline & Prerequisites)

要驱动上述技术引擎，我们接单时（或者准备副业 Demo 时），绝不能再向客户要“几张生活照”，而是要按照以下工业标准准备 **4 大类核心资产** ：

### 1. 连续动作序列帧（最重要的重武器，占最终效果 70%）

必须准备  **3 ~ 4 段特定动作的透明短片（每段 3 ~ 5 秒）** ，后续由技术转化为连续图片：

* **Hero Loop (首屏定格唤醒)：** 正面打哈欠、伸懒腰、或歪头卖萌。用于打开网页的第一秒抓住眼球。
* **Transition (场景过渡位移)：** 小猫向前扑、跑酷、或者钻进宠物包/行李箱。用于页面从上一屏切换到下一屏时的连接。
* **Interaction (功能特性展示)：** 翻肚皮睡觉（对应懒惰人设）、爪子划拉平板电脑（对应聪明人设）、低头喝水。
* **规格要求：** 必须是**背景干净**的视频，最终导出的单张序列帧分辨率建议为 1080px 宽度，格式必须为  **WebP 带有 Alpha 透明通道** （为了压缩体积，保证极速加载）。

### 2. 空间视差静态切图（用于 Z 轴前后夹层）

* **全身立绘高清图 (2~3张)：** 坐姿、躺姿等。要仔细抠掉背景，连胡须和边缘毛发都要保留。
* **互动中心特写大头贴 (1张)：** 500x500 像素以上，正脸微笑，用于网页右下角常驻的“悬浮云撸猫”挂件入口。
* **前景遮挡物切图 (选配)：** 猫粮袋子、沙发前沿、地毯边缘。网页滚动时，这些物体可以在文字或宠物前方漂浮，制造强烈的 3D 景深感。

### 3. 微交互与游戏化图形 (UI / FX Elements)

* **专属印章与徽章：** 视频中出现的“表扬头子”、“全自动闯祸机”、“24小时营业”等趣味设计图标（使用 Figma 绘制成纯 SVG）。
* **物理粒子素材 (128x128px PNG)：** 💖 爱心、🐟 小鱼干、🐾 猫爪印、🪙 金币、Zzz 沉睡气泡。当用户在右下角疯狂点击“撸猫”和“喂食”时，前端使用 Canvas API 实现粒子从右下角向全屏喷射并伴随重力下落的特效。

### 4. 杂志风情绪文案与人设词条

废弃传统的“品种/体重/年龄”流水账，改用对立冲突感强烈的时尚杂志文案：

* **RPG 属性面板：** 睡眠深度 (99%)、拆家战力 (MAX)、掉毛指数 (一年掉两斤)、黏人程度 (零下10度)。
* **搞怪罪状与个性：** “比起窝，更爱住盒子和行李箱”、“上辈子是个水桶，爱玩水爱喝水”。
* **情感共鸣金句：** “笑起来是好看，但是掉毛啊”、“随地撒落全地阳光”、“对对对，喜欢白天睡觉晚上行动”。

## 四、 零成本 AI 素材生成工作流 (AI Asset Pipeline)

作为副业，我们不可能每次接单都带着绿幕去客户家拍视频。为了实现 **高毛利、快交付** ，我们可以利用当下的“AI 工具链”无中生有，将普通照片转化为顶级序列帧：

```
[步骤1: 定妆] Midjourney/Flux 生成宠物完美立绘
     ↓
[步骤2: 动态] Kling(可灵) / Luma 输入提示词生成 5 秒动作短片
     ↓
[步骤3: 抠图] Runway / BiRefNet 自动精准去除背景
     ↓
[步骤4: 抽帧] FFmpeg 脚本一键切分为 100 张透明 WebP 序列帧
     ↓
[步骤5: 部署] 放入 Next.js `/public/sequence/` 目录，交由 GSAP 渲染！
```

* **详细操作策略：**
  * **拿到客户的 1-2 张普通真猫照片** -> 喂给 Midjourney 使用 `--cref` (角色一致性) 重新生成光影极佳的纯色背景大图。
  * 将这张静止大图传入  **快手可灵 (Kling) 或 Luma** ，写提示词： *“A cute fluffy cat opening mouth to yawn, smooth cinematic motion, studio light”* ，生成 4 秒完美短片。
  * 用开源工具将短片以  **24 fps 抽帧** ，打包为静态资源。这个过程咱们不需要敲一行代码，全靠 AI 自动流完成，**单站素材准备成本压缩在 1 小时以内！**
